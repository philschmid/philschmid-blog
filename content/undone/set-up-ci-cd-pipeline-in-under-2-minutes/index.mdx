A quick Tutorial how to set up a CI/CD Pipeline with AWS Codebuild for AWS Lambda in under 2 Minutes

Vorraussetzungen:

* serverless framework installiert
* aws account set up
* aws lambda function (i use a simple hello world and then hello ci cd )
* Github Account


tl;DR;

Copy buildspec.yaml to root directory. 
Create AWS folder on root level add codebuild.yaml file. 
ADD `custom: GITHUBREPO` in serverless.yaml. 
Run serverless deploy.


Man braucht ein buildspec.yaml darin wird beschrieben, was in codebuild ausgeführt werden soll.
Codebuild configuration werden über cloudformation/ in serverless.yaml mit integriert über file import
erstes mal manuel deploy, so werden die codebuild project angelegt.
